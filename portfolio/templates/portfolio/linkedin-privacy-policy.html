1. Information We Collect

We do not collect personal data from visitors to our website. The following information is collected and processed solely for the purpose of displaying the owner’s LinkedIn posts:

  • LinkedIn Post Data: We use the LinkedIn API to retrieve the website owner's most recent LinkedIn posts. This data includes the content of posts and links to LinkedIn, and is displayed publicly on the website.
  • Usage Data: We may collect basic information such as IP addresses, browser type, and usage patterns (e.g., which pages were visited) to improve the performance and usability of the website.

2. How We Use the Information

We use the information we collect in the following ways:

  • To Display LinkedIn Posts: We use the LinkedIn API to authenticate with the website owner's LinkedIn account and display the two most recent posts on the website. No visitor authentication is required or requested.

3. Sharing Your Information

As this website does not collect personal information from its visitors, no personal data is shared with third parties. However, the following details apply to the data used to display LinkedIn posts:

  • Service Providers: We may use third-party services to host the website or enhance functionality (e.g., cloud hosting or content delivery networks). These services may have access to basic usage data but do not access any LinkedIn data or personal visitor information.
  • LinkedIn API: The website interacts with LinkedIn’s API to display the owner’s posts. No other information is shared with LinkedIn except for the website owner’s credentials, which are used solely to retrieve the posts.

4. Data Retention

We do not store any personal data from visitors. Data related to the website owner's LinkedIn posts is updated periodically and refreshed according to LinkedIn’s API guidelines. No user information is retained or stored on the website.
